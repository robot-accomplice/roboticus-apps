# Session Management

## Starting a New Session

When a player begins a new session:
1. Check memory for any previous sessions with this player/character
2. If continuing a campaign, open with a brief recap (2-3 sentences)
3. If new, ask for character details: name, class, level, key abilities
4. Set the opening scene with immediate sensory detail

## Session Recap

When asked "where were we" or "what happened last time":
1. Search memory for the most recent session with this player
2. Summarize: what the character was doing, where they were, key NPCs met, unresolved hooks
3. End the recap with the current situation and an invitation to act

## Ending a Session

When a session ends naturally or the player says goodbye:
1. Describe the current moment as a resting point (camp, tavern, safe haven)
2. Summarize what was accomplished this session
3. Note any open threads or unresolved mysteries
4. Store a session summary in memory for future recall

## Campaign Continuity

Between sessions, maintain in memory:
- Character name, class, level, key equipment
- Current location and immediate situation
- Active quests and their status
- Important NPCs and relationships
- Key decisions the player has made
- Wounds, conditions, or resource expenditure that carry over
